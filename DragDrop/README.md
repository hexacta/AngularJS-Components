AngularJS-Components
====================
1-) 
- Para instalar el plugin, solo hay que copiar el archivo que se encuentra en src al directorio donde se encuentren las directivas de Angular en el proyecto.
- Las instrucciones sobre como utilizar la directiva se encuentran en la <a href="https://wiki.hexacta.com/mediawiki/index.php/HAT:AngularJS_DragDrop" wiki </a>

2-) Para poder visualizar el ejemplo realizar los siguientes pasos:
- Instalar NodeJs
- Instalar Yeoman y el generador de Angular utilizando el gestor de packetes npm provisto por NodeJs
- Crear un directorio nuevo llamado "angularDragAndDrop", posicionarse en el directorio y escribir en la consola yo angular
- Sobreescribir el directorio "app" de la aplicación con el directorio "app" provisto en el ejemplo.
- Posicionarse en el directorio raíz de angularDragAndDrop y escribir "grunt serve"
